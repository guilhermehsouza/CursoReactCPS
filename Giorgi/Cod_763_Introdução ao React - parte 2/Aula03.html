Uma das premissas do React é que você incorpore todo código HTML através do JSX.

Normalmente, em projetos relacionados ao front-end, mantemos o código em HTML, CSS e JavaScript em arquivos separados. No React, porém, isso funciona de um modo um pouco diferente.

Nos projetos em React, não criamos arquivos em HTML separados, pois o JSX (JavaScript XML) nos permite escrever uma combinação de HTML e JavaScript em um mesmo arquivo. No entanto, é possível separar seu CSS em outro arquivo.
