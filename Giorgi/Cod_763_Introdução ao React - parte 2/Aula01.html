Aula 1 - Criando e entendendo um novo Projeto

Para iniciarmos nosso curso são necessários 3 Pré-requisitos.

  1. Ter feito o curso de Introdução ao React parte-1 ou já ter conhecimentos básicos em React.
  2. Verificar a instalação do Node.js
  3. Verificar se o npm e npx estão funcionando corretamente.

Para isso execute os comandos no prompt ou terminal do VS Code conforme imagem abaixo.

Se tudo deu certo, então vamos criar um novo projeto. Para isso é importante que crie uma pasta onde o projeto será armazenado, no meu caso vou criar dentro da pasta dos arquivos deste curso.

Para criar um novo projeto execute o código: npx create-react-app my-app-react e aguarde alguns minutinhos.

Após criar o projeto, acesse a pasta my-app-react e execute o comando npm start , com isso seu navegador deve ser aberto com a página inicial do React.

Note que será criada a pasta my-app-react.

Relembrando a estrutura das pastas

  • README.md é um arquivo em markdown que inclui muitas dicas úteis e links que podem ajudar você em seu aprendizado no uso do Create React App.
  • node_modules é uma pasta que inclui todo o código relacionado às dependências que o Create React App instalou. Você dificilmente precisará entrar nessa pasta.
  • package.json é o arquivo que gerencia as dependências da aplicação e o que está incluído na pasta node_modules do projeto, além dos scripts de que precisamos para executar a nossa aplicação.
  • .gitignore é um arquivo usado para excluir os arquivos e pastas do rastreamento do Git. Não precisamos incluir pastas grandes, como a pasta node_modules, por exemplo
  • public é uma pasta que usaremos para armazenar nossos itens estáticos, como as imagens, svgs e fontes da aplicação em React.
  • src é a pasta que contém o código-fonte. É lá que todo nosso código relacionado ao React estará e é nela que trabalharemos fundamentalmente na criação de nossa aplicação.

A pasta src

Esta pasta é onde os principais arquivos do React estão armazenados, vamos identifica-los.

- O arquivo App.js é o arquivo que contém o primeiro componente deste projeto, é onde serão feitos os nossos exemplos.

- O arquivo index.js é o arquivo que contém toda a estrutura do React, é ele quem renderiza o arquivo App.js e qualquer outro necessário, inclusive é nele que se encontra a função ReactDOM. Quando é executado o npm start este arquivo é chamado primeiro, ele renderiza o conteúdo do arquivo App.js e exibe seu conteúdo pelo arquivo public/index.html. Inicialmente são os três arquivos envolvidos na execução da página inicial do React.

Vamos visualizar o código destes arquivos

Arquivo index.js

Este arquivo faz a importação das dependências necessárias no inicio do arquivo e renderização do App.js. Nem todos os arquivos importados são necessários, os arquivos css e reportWebVitals podem ser removidos ou alterados se necessário.

Arquivo app.js

Este arquivo é onde por padrão é gerado o primeiro componente do projeto.

Arquivo public/index.html

É o arquivo index padrão, ele recebe e exibe o resultado dos componetes React da página App.js.
