Neste segundo exemplo iremos criar uma variável simples e na sequência exibi-la usando o JSX.

JSX é uma extensão da linguagem JavaScript que é usada pelo React para criar interfaces de usuário. Ele permite que você misture código JavaScript com sintaxe de HTML, o que torna mais fácil escrever componentes de interface de usuário em um único arquivo de código. Se você está gostando de aprender React é essencial aprender JSX.

Estrutura do Componente

Um componente é um bloco de código reutilizável e independente, que divide a interface do usuário em partes menores.

Para conseguir usar um componente mais tarde, você precisa, primeiramente exportá-lo de modo a importá-lo em outro lugar

Como pode ser notado no exemplo principal e nos dois exemplos que criamos é que a estrutura básica de componente do React é com function e return

Note que você pode separar o export default da função, como é feito no app.js original.

Importando o componente

Desse modo, um componente funcional em React:

  • é uma função em JavaScript/ES6
  • deve retornar um elemento em React (JSX)
  • sempre começa com letra maiúscula (convenção dos nomes)
  • aceita props como parâmetro, se necessário

Não esqueça !!!

No projeto padrão o conteúdo do arquivo App.js é renderizado pelo index.js e seu resultado é exibido pela index.html através da função ReactDOM
