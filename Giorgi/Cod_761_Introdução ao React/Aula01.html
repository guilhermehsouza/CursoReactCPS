Aula 1 - Instalando o Node.JS

Depois de uma pequena introdução chegou hora preparar o ambiente de testes. Como qualquer framework ou linguagem alguns requisitos são necessários, então vamos lá.

Lembrando que neste curso foi utilizado o SO Windows 10.

React é uma biblioteca/framework JavaScript para criar aplicações que rodam no navegador, porém os pacotes que iremos utilizar durante o desenvolvimento serão gerenciados pelo Node Package Manager (NPM), inclusive o próprio React. Começaremos então indo até o site oficial do Node.js e iremos instalar sua versão LTS.

Pré-requisito: Antes de instalar ou começar a trabalhar no React, temos que ter certeza de que algumas coisas estão disponíveis em nosso sistema.

  • Qualquer editor de texto (neste curso foi utilizado o VS Code)
  • NodeJS instalado em seu sistema:
    1. Download Node.js para Windows
    2. Versão utilizada no curso: node-v18.17.1-x64

  • Qualquer dúvida utilize os tutoriais abaixo.
    1. Instalação de Node.js no Windows
    2. Instalação de Node.js no Linux

A instalação do Node.js não possui nenhuma particularidade, pode ser feita a instalação padrão.

Após concluir a instalação, verifique se foi realizada com sucesso. Abra o prompt de comando e digite node -v
