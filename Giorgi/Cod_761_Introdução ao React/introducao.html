Introdução - O que é o React?

React é uma biblioteca para interfaces de usuário web e nativas. 

O React permite construir interfaces de usuário a partir de peças individuais chamadas componentes. É possível criar seus próprios componentes React e em seguida, combina-los em telas, páginas e aplicativos inteiros.

Surgimento:

React foi criado pelo engenheiro de software Jordan Walke em 2011, enquanto trabalhava no Facebook. A equipe do Facebook estava procurando uma maneira de tornar mais fácil e mais eficiente o desenvolvimento de interfaces de usuário, e o React surgiu como uma solução para esse problema. Desde então, o React se tornou uma das bibliotecas de interface de usuário mais populares e amplamente utilizadas no mercado, sendo usado por empresas de todos os tamanhos, incluindo o Facebook, Netflix, Airbnb, entre outras. 

  • 2011: Facebook começa a usar XHP, uma biblioteca de interface de usuário baseada em HTML, para construir seus aplicativos.
  • 2013: Facebook lança o React como código aberto.
  • 2015: React Native é lançado, permitindo o desenvolvimento de aplicativos móveis nativos com React.
  • 2016: Facebook lança o React VR, uma biblioteca para desenvolver aplicativos de realidade virtual com React.
  • 2019: A v16.8 é lançada com suporte ao React Hooks, que permite aos desenvolvedores gerenciar o estado e os efeitos do React de forma mais fácil e direta
  • 2021: Liberado novas funcionalidades que virão no React 18.

Como funciona? 

O React é uma biblioteca que utiliza a linguagem de programação JavaScript. O JS é uma das mais versáteis e populares linguagens de programação do mundo e conta com um grande número de bibliotecas e outras tecnologias que a utilizam. Entre elas, Node.js, Svelte, Angular, VueJS, jQuery, Ember.js e, é claro, o React.

O uso de componentes é um dos pilares de sustentação do React. Para compreender e agir sobre uma determinada página, o React a quebra em pequenos pedaços, chamados de componentes, e os isola, sendo cada um deles independentes entre si perante a tecnologia. Estes componentes são reutilizáveis e, uma vez categorizados, podem ser reaproveitados em quaisquer páginas de um site ou aplicativo.

Virtual DOM

O React utiliza um Virtual DOM, ou VDOM, é uma representação em memória de um DOM (Document Object Model) real de uma interface. Na prática, DOM real é a estrutura que representa a camada visual de uma página.

Em vez de manipular diretamente o DOM do navegador, o React cria um DOM virtual na memória, onde faz todas as manipulações necessárias, antes de fazer as alterações no DOM do navegador. Com isso o React descobre quais alterações foram feitas e altera apenas o que precisa ser alterado.

JSX

A sintaxe utilizada pelo React é chamada JSX, que significa JavaScript Syntax Extension. Trata-se de uma extensão para o JavaScript, ou seja, oferece algumas funcionalidades a mais e permite escrever de forma simples JavaScript e HTML no mesmo código. Sua característica de mesclagem entre JavaScript e HTML oferece uma sintaxe de fácil aprendizado, além de facilitar o desenvolvimento e o entendimento das funcionalidades do código.

Por que usar React?

  • Fácil de escrever e usar
  • Melhor desempenho com Virtual DOM
  • Amigável para SEO (Search Engine Optimization)
  • Suporte a Componente Reusável em JavaScript
