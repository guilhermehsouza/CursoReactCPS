Aula 2 - Criando um projeto com React

Existem algumas formas de criar um projeto em React.

  • utilizando códigos cdn
  • utilizando o Next.js

    • Next.js é uma estrutura React flexível que fornece blocos de construção para criar aplicativos da web repidamente.

    • saiba mais sobre o Next.js neste link
  • utilizando o Create React App - será utilizado neste no curso

    • é um conjunto de ferramentas e funcionalidades pré-configuradas para que você possa iniciar um projeto React.

    • Normalmente este pacote é indicado aos iniciantes no aprendizado de React para que possam ir direto ao código e testar seus conhecimentos.

    • saiba mais sobre o CRA neste link

Para iniciarmos, crie uma pasta em seu computador para armazenar a estrutura do projeto.

Neste exemplo a pasta será criada na mesma pasta deste Projeto.

Não esqueça de acessar a pasta no terminal do VS Code, é nela que o projeto será criado.

Usando o Create React App

Para criar um novo projeto em React, podemos usar a ferramenta npx, contanto que você tenha, no mínimo, a versão 5.2 do npm.

Observação: você pode conferir qual versão do npm você tem digitando em seu terminal npm -v

O npx nos dá a capacidade de usar o pacote create-react-app sem ter de instalá-lo primeiro, o que é muito conveniente.

Usar o npx também garante que estejamos usando a versão mais recente do Create React App para criar nosso projeto:

Digite o seguinte comando no terminal do VS Code: npx create-react-app meu-app-do-react

Quando executarmos esse comando, uma pasta chamada "meu-app-do-react" será criada no nosso computador, na pasta onde especificamos e todos os pacotes necessários serão instalados automaticamente.

Observação: a criação de uma nova aplicação do React com create-react-app geralmente leva de 2 a 3 minutos, às vezes mais.

O Create React App também nos dá alguns modelos de uso para tipos específicos de projetos do React.

Se a instalação transcorrer sem problemas, deverá apresentar o resultado abaixo.

  • Na sequencia, execute os seguintes comandos:

    1. cd meu-app-do-react - comando para acessar a pasta do projeto

    2. npm start - comando para inicializar o projeto React.

Resultado esperado do npm start.

- Se tudo correu bem até aqui, PARABÉNS !!! Seu projeto deve ter sido aberto em um navegador.

IMPORTANTE - caso você feche o VS Code é preciso executar o comando npm start na pasta do projeto novamente.
